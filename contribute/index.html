02

Open an issue or pull request. Codex reviews evidence, tests the exact commit, and owns integration into main.

03

Receive access

A maintainer grants a GitHub repository role. Control access starts at triage and is rechecked on every protected request.

Never send an xAI API key to this website. This site is static, holds no credentials, and requests nothing sensitive. Contributor sign-in happens only at control.grokmcp.org.
